Copy these icons into place:

[wikipedia](https://en.wikipedia.org/wiki/Obsidian_(software)#/media/File:2023_Obsidian_logo.svg)

```
~/.local/share/icons/hicolor/256x256/apps/co.anysphere.cursor.png
```
